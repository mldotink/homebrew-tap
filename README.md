# Homebrew Tap for Ink

Homebrew formulae for [Ink](https://ml.ink) — the cloud platform for deploying and managing services.

## Install

```bash
brew install mldotink/tap/ink
```

## Usage

```bash
ink login
ink deploy my-app --repo my-repo --port 3000
ink list
ink status my-app
ink logs my-app
```

For more information, visit [ml.ink](https://ml.ink).
